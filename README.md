#### Running:

Edit package.json 

##### Weather

 - add your http://openweathermap.org api key
 - add location

##### News

 - add your https://newsapi.org/ api key


```
npm install
node app.js <argument>
```
 
 
#### Add alias:

```
alias weather="<path-to-directory>/app.js weather"
alias news="<path-to-directory>/app.js news"
```
